# Book Scraper

A Python web scraper that extracts book data from Books to Scrape.

## What it extracts
- Title
- Price
- Rating

## Output
Saves all data to a `books.csv` file that can be opened in Excel.

## How to use
Install dependencies:
pip install -r requirements.txt

Run the scraper:
python main.py

## Requirements
Python 3.x
